# Saarthi AI 🤖

Saarthi AI is a voice-enabled intelligent chatbot built using Python, Machine Learning, and NLP. It supports both Hindi and English conversations and works in offline as well as hybrid AI modes.

---

## 🚀 Features

- 🎤 Voice-based interaction
- 🌐 Hindi ↔ English support
- 🧠 Context-aware responses
- ⚡ Offline + Hybrid AI system
- 🤖 ChatGPT-style assistant

---

## 🛠 Tech Stack

- Python
- Machine Learning
- NLP
- Eel (Frontend + Backend bridge)
- SpeechRecognition
- PyAudio
